**Create `src/whatsapp/authState.ts`:**

Import the following from `@whiskeysockets/baileys`:
- `proto`, `initAuthCreds`, `BufferJSON`, `makeCacheableSignalKeyStore`
- Types: `AuthenticationState`, `SignalDataSet`

Import `pool` from `../db/pool` and type `Logger` from `pino`.

Export `async function usePgAuthState(repId: string, logger?: Logger)` returning `Promise<{ state: AuthenticationState; saveCreds: () => Promise<void> }>`.

**Implementation steps inside usePgAuthState:**

1. **Load or initialize creds:**
   - Query: `SELECT creds FROM wa_auth_creds WHERE rep_id = $1` with `[repId]`
   - If row exists: parse with `JSON.parse(JSON.stringify(credsRow.rows[0].creds), BufferJSON.reviver)`
   - If no row: call `initAuthCreds()`

2. **Build raw SignalKeyStore object** with two methods:

   `async get(type: string, ids: string[])`:
   - Query: `SELECT key_id, value FROM wa_auth_keys WHERE rep_id = $1 AND key_type = $2 AND key_id = ANY($3)` with `[repId, type, ids]`
   - Build `data: Record<string, any> = {}`
   - For each row: deserialize with `JSON.parse(JSON.stringify(row.value), BufferJSON.reviver)`
   - CRITICAL: If `type === 'app-state-sync-key'` and value is truthy, wrap with `proto.Message.AppStateSyncKeyData.fromObject(value)`
   - Set `data[row.key_id] = value`
   - Return `data`

   `async set(data: SignalDataSet)`:
   - Acquire client via `const client = await pool.connect()`
   - Wrap in try/catch/finally with `BEGIN`/`COMMIT`/`ROLLBACK`/`client.release()`
   - Iterate `Object.entries(data)` as `[type, keys]`
   - For each `[id, value]` in `Object.entries(keys ?? {})`:
     - If `value == null`: DELETE from wa_auth_keys WHERE rep_id=$1 AND key_type=$2 AND key_id=$3
     - Else: serialize with `JSON.parse(JSON.stringify(value, BufferJSON.replacer))`, then UPSERT:
       `INSERT INTO wa_auth_keys (rep_id, key_type, key_id, value) VALUES ($1, $2, $3, $4) ON CONFLICT (rep_id, key_type, key_id) DO UPDATE SET value = EXCLUDED.value`

3. **Wrap with cache layer:**
   - `const keys = makeCacheableSignalKeyStore(rawStore, logger)`

4. **Define saveCreds:**
   - Serialize creds: `JSON.parse(JSON.stringify(creds, BufferJSON.replacer))`
   - UPSERT: `INSERT INTO wa_auth_creds (rep_id, creds, updated_at) VALUES ($1, $2, NOW()) ON CONFLICT (rep_id) DO UPDATE SET creds = EXCLUDED.creds, updated_at = NOW()`

5. **Return** `{ state: { creds, keys }, saveCreds }`

**SECURITY:** All SQL uses parameterized queries ($1, $2, $3) — never interpolate repId or any value into SQL strings.
